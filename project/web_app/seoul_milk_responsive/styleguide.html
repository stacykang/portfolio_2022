Overview

서울우유는 국내 우유 시장 1위 기업으로, 설립 이래 80년간 고객의 건강을 생각하며 낙농업을 선도해 왔습니다.
공식 쇼핑몰인 '나 100샵'을 운영하고 있으며, 가정배달 서비스를 제공하고 있습니다.

기존 웹사이트 접속 시 소개 영상과 SNS 콘텐츠 등이 로딩되는 시간이 오래 걸린다는 문제가 있었습니다.
따라서 영상 길이를 조정하여 용량을 최적화하고 외부링크로 연결되는 방식을 수정하여 로딩시간을 단축하였습니다.

기존 사이트의 복잡한 레이아웃을 전체적으로 깔끔하게 재구성하였고 서울우유의 대표 컬러를 강조해야하는 부분에 활용하였습니다.
브랜드 서체인 서울우유체를 본문에도 사용시 가독성이 떨어지는 문제가 있어 제목에만 사용하여 가독성을 높였습니다.
